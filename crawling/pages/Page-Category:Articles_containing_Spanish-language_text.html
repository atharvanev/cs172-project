This category contains articles with Spanish-language text. The primary purpose of these categories is to facilitate manual or automated checking of text in other languages.

This category should only be added with the {{Lang}} family of templates, never explicitly.

For example {{Lang|es|¿Habla español?}}, which wraps the text with <span lang="es">. Also available is {{Langx|es|¿Habla español?}} which displays as Spanish: ¿Habla español?.
